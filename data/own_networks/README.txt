# DawnRank Network

This is the full version of the same network used for the original dawnrank publication. (http://genome.compbio.cs.cmu.edu/~jianma/software/DawnRank/fullNetwork.rda)

# OncoImpact Netwok

OncoImpact uses a functional interaction network generated by Wu G et al (https://genomebiology.biomedcentral.com/articles/10.1186/gb-2010-11-5-r53). The version downloaded here is the most recent version (v070323), NOT the original version used in the OncoImpact publication.


# PRODIGY and PersonaDrive Network

Both PRODIGY and PersonaDrive use versions of the STRING network, filtered to edge weights > 0.7. This network is the latest version of the STRING network (v12.0) (https://stringdb-downloads.org/download/protein.links.v12.0/9606.protein.links.v12.0.txt.gz), with the STRINGids converted to hugo symbols using the aliases file.
